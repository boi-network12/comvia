````md
# ✨ Overview

Comvia is a modern customer communication platform that enables businesses to connect with website visitors instantly through live chat.

Unlike traditional customer support platforms that restrict customization, Comvia is designed with developers in mind—giving you complete control over your widget's appearance while keeping integration as simple as adding a single script.

Whether you're building a startup, SaaS product, e-commerce store, portfolio, or enterprise application, Comvia helps you provide a better support experience.

# 🚀 Getting Started

Clone the repository.

```bash

Go into the project.

```bash

# 💬 Widget Installation

The Comvia widget is designed to be integrated within seconds.

Simply include the script on your website.

```html
<script
    src="https://comvia-widget.vercel.app/comvia-widget.min.js"
    data-company-id="YOUR_COMPANY_ID"
    data-position="bottom-right"
    data-color="#F97316"
    data-icon="chat"
    data-company-name="Comvia Support"
></script>
```

More installation methods and SDK documentation are coming soon.

---

# 🧩 Widget Customization

Customize your widget using data attributes.

| Attribute | Description |
|------------|-------------|
| data-company-id | Workspace ID |
| data-position | Widget position |
| data-color | Primary color |
| data-icon | Widget icon |
| data-company-name | Company name |
| data-company-logo | Company logo |
````
